How it works

0

Problem → Error Function

Any problem becomes: minimize E(x). The error measures how wrong you are.

E(x) = (x − 5)²
1

Variable → Frequency

Each variable becomes an oscillator. Its value determines its pitch.

f(x) = 200 + 20x
2

Error → Amplitude + Noise

High error = quiet + noisy. Low error = loud + clean. The math guarantees it.

A = exp(−E(x))   noise ∝ E(x)
3

Gradient → Vibrato

Large gradient = frequency shakes. At the minimum, gradient = 0, tone is rock-steady.

vibrato ∝ |∇E(x)|
4

Multi-variable → Interference

Multiple variables = multiple oscillators. Constraint violation = beating between tones.

s(t) = Σ sin(fᵢ t)
